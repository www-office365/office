How to Download Microsoft Office

A quick guide for Windows and Mac users

For Windows Users

  1. Go to www.office.com
  2. Click Sign In and log in with your Microsoft account.
  3. Click Install Office.
  4. Choose Office 365 apps and download begins.
  5. Run the downloaded installer and follow prompts.

For Mac Users

  1. Visit www.office.com
  2. Sign in with your Microsoft account.
  3. Click Install Office.
  4. Select the Mac version and start the download.
  5. Open the downloaded `.pkg` file and install.

Need Help?

Visit: support.microsoft.com/office
